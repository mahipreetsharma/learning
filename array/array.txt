Array
    array is the collection of same data types values.
    int a[5]; this array a will store only 5 values it 
    the index of array start from 0 

Declaration of array
    int A[5]; //only declared so there will garbage values stored in array
    int A[5]= {1,2,3,4,5} // declaration with initialization
    int A[5] = {1,2} //all element except on 0 and 1 index will zero, and 0 initilize with 1, and 1 initilize with 2
    int A[] = {1,2,3,4,5,6}; if there is no predefined size 

Access of values stored in array
    COUT<<a[0]; PRINT value stored at array a at index 0;

Static vs Dynamic array
    Static array
        if the size of array is predefined while compile time in c
        size of array cannot be updated after decclaration

    Dynamic array
        the size of the array is defined while runtime
    
Array in heap
    int a[5]
    int *p;
    p = new int[5];
    this create the memory for the 5 integers in heap memory and can be accesed through the pointer.
