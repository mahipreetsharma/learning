Recursion
    When a function call itself then it is called recursion
    Inside a recursive function there should be a condition which ends the recursion
    Recursion function is traced in the tree form

example 
    void fun(int n){
        if(n>0){
            printf("%d",n);
            fun(n-1);
        }
    }
    void main(){
        int x = 3;
        fun(x)
    }

    tree representation
    fun 3
    3---fun(2)
        2---fun(1)
            1---fun(0)
                X
                output will 3 2 1 ;

    Recursion saves in stack form

Time complexity
    according to the input iteration value , n

Types of recursion
tail recursion
head recursion
tree recursion
inderect recursion
nested recursion

    Tail Recursion
        It should not perform any task while returning time.
    Head Recursion
        If all processing is done after the function call. all performing are done while reeturning time
    Tree recursion
        If there is more than one function call.
    Indrect recursion
        if function call is not directly, the call is in cycle using multiple funtions.
    Nested Recursion
        When function call itself as a function fun(fun(n-1))
